Просто поклади потрібні файли сюди і вкажи шлях
в налаштуваннях модуля.

ФОРМАТИ: MP3, OGG, WAV (MP3 рекомендовано)
ШИФРУВАННЯ: не потрібне — Foundry грає звичайні файли

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ФАЙЛИ (назви рекомендовані, але можна будь-які):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  impact.mp3    ← грає під час impact-кадру та появи назви кидку
                  Рекомендовано: 2-4 сек, драматично, барабани

  ambient.mp3   ← фоновий ембіент поки гравці роблять кидки
                  Рекомендовано: 30-120 сек, loop, напружено

  success.mp3   ← фанфара при оголошенні УСПІХУ
                  Рекомендовано: 3-6 сек, переможно

  failure.mp3   ← мелодія при оголошенні ПРОВАЛУ
                  Рекомендовано: 3-6 сек, похмуро

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
ЯК ПІДКЛЮЧИТИ:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Foundry → Game Settings → Module Settings
         → Епічні Кидки UA

Вводи шлях до файлу, наприклад:
  modules/epic-rolls-ua/sounds/impact.mp3
  modules/epic-rolls-ua/sounds/ambient.mp3
  modules/epic-rolls-ua/sounds/success.mp3
  modules/epic-rolls-ua/sounds/failure.mp3

Або будь-який URL з інтернету (https://...).
